# youtube-data-analysis
Exploratory Data Analysis (EDA) of a YouTube dataset using Python, including statistical analysis and data visualization to understand video performance, engagement patterns, and content trends.

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a YouTube dataset to understand trends in video performance and audience engagement. The analysis includes both non-graphical statistical analysis and graphical data visualization to uncover insights about how different factors influence video success.

## Dataset Description
The dataset contains information about YouTube videos including video titles, channels, categories, languages, countries, views, likes, dislikes, comments, duration, and upload dates.

## Analysis Performed

### Non-Graphical Analysis
- Counting unique videos, channels, and owners
- Category and language distribution
- Country-wise video distribution
- Video upload trends by year
- Statistical measures such as mean, minimum, and maximum views
- Engagement metrics like like-to-dislike ratio

### Graphical Analysis
- Bar charts for category engagement
- Pie charts for language distribution
- Scatter plots for views vs likes correlation
- Histograms for video duration distribution
- Heatmaps showing correlation between engagement metrics
- Boxplots and violin plots for category comparison

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Entertainment category generates higher engagement.
- Views show strong correlation with likes and comments.
- Longer videos do not always result in higher views.
- Engagement metrics provide better performance insights than views alone.
